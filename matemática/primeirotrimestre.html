Sistemas de númeração

É uma organizazação de simbolos, que representam quantidades númericas. Sendo eles:

  • Sistema binario;
  • Sistema octal;
  • Sistema hexadecimal;


Sistema Binário

O sistema binário é composto por dois simbolos, os números "0" e "1"
